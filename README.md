SquirrelID
==========

SquirrelID is a library for player UUID-related tasks involving Minecraft and
Mojang properties.

SquirrelID is open source and is available under the GNU Lesser General Public
License v3.

Compiling
---------

Use Maven 3 to compile SquirrelID.

    mvn clean package
